Description

This a software timer driver, there is 3 way of use it:

  • as a counter
  • for time out
  • With a call back event It's need a main hardware or system periodic timer to work

use

This is an example of the use of a time out timer

Note:
the return values are not tested to simplify 
        DRV_SoftTimer_Cfg sParam;
        DRV_SoftTimer_Handle hTimer1=NULL;

        DRV_SoftTimer_Init();
        //config parameters
        sParam.tType=TimeOut;
        sParam.uiValue = 10;
        //open the timer
        DRV_SoftTimer_Open("timer1",&hTimer1,&sParam)==No_Error)
        //start the timer
        DRV_SoftTimer_Start(hTimer1);
        //do something
         ...
        //test the timer value
        if( DRV_SoftTimer_GetValue(hTimer1) == 0 )
          printf("Timeout!\n");
        //close the timer
        DRV_SoftTimer_Close(hTimer1);

        DRV_SoftTimer_Terminate();
